# Выбор локации для скважины

## Введение

Добро пожаловать в проект "Выбор локации для скважины". 
В рамках данного проекта мы будем работать с добывающей компанией "ГлавРосГосНефть" и решать важную задачу – определение наилучшей локации для бурения новой нефтяной скважины.

У нас есть данные о пробах нефти в трёх различных регионах. 
В каждом из этих регионов у нас есть информация о 10,000 месторождениях, включая качество нефти и объём её запасов. Наша цель – построить модель машинного обучения, которая поможет нам определить регион, в котором добыча нефти принесёт наибольшую прибыль.

Для этого мы будем следовать определённым шагам:

1. В избранном регионе мы будем искать месторождения и для каждого из них определять значения различных признаков.
2. Затем мы построим модель для оценки объёма запасов нефти.
3. На следующем этапе мы выберем месторождения с самыми высокими оценками значений. Количество месторождений, которые мы сможем разработать, будет зависеть от бюджета компании и стоимости разработки каждой скважины.
4. И, наконец, наша прибыль будет равна суммарной прибыли отобранных месторождений.

Для анализа возможной прибыли и рисков мы будем использовать методику Bootstrap, что позволит нам более точно оценить потенциальные результаты и принять обоснованные решения.

Данный проект будет ориентирован на максимизацию прибыли для компании "ГлавРосГосНефть" и снижение рисков при выборе локации для бурения новой нефтяной скважины.
